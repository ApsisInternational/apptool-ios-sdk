  • +setMinimumLogLevel:

    Sets minimum log level. (default: ONELogLevelNone) Available levels: ONELogLevelNone, ONELogLevelError, ONELogLevelWarning, ONELogLevelInfo. Each subsequent level contains previous log event. E.g. ONELogLevelError contains only Error log events, and ONELogLevelInfo contains Error, Warnings and Info log events.
